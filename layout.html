Magazine Layout Design

This magazine layout was an exercise to use grids creatively. My goal was to create a layout that stuck to a set of rules, but it was loose enough to be interesting.

Using bold images and negative space makes the layout more dynamic. In some cases there is more negative space than text on the page, and this was my way of really adding contrast. It makes the articles more readable because it's not just a solid block of text.
